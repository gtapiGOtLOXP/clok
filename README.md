# 🎮 Projects

---

## 🟩 Minecraft (Python)

A simple Minecraft-style project written in Python.

### 📁 Location
```
minecraft/minecraft.py
```

### ⚙ Requirements
- Python **3.13+**
- The file **`grass.png`** must be in the same folder as `minecraft.py`.

### ▶ Run

```bash
cd minecraft
python minecraft.py
```

Folder example:

```
minecraft/
│
├── minecraft.py
└── grass.png
```

## 🟦 Pacman (HTML + JavaScript)

A simple **Pac-Man browser game**.

### 📁 Location

```
pacmanhtmljs/pacman.html
```

### ▶ Run

Just **open the file** in your browser.

```
pacmanhtmljs/pacman.html
```

or double click it.

### 🎮 Controls

| Key | Action |
|----|----|
| W / ↑ | Move Up |
| A / ← | Move Left |
| S / ↓ | Move Down |
| D / → | Move Right |
